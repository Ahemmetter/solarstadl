Photovoltaik-Anlagen

Photovoltaik-Anlagen dienen zur Erzeugung von Strom - im Unterschied zu den Sonnenkollektoren-Anlagen, welche zur Erzeugung von Warmwasser verwendet werden. Die Solar-Stadl Hemmetter GmbH hat sich seit vielen Jahren auf die Installation von Photovoltaik-Anlagen spezialisiert. Die Anlagen können auf die unterschiedlichsten Gegebenheiten abgestimmt werden. Ob Kleinanlage auf dem Privathaus oder Grossanlage auf dem Dach eines landwirtschaftlichen Gebäudes oder im Freiland - unsere Photovoltaik-Anlagen werden auf Sie speziell ausgelegt, geplant und montiert.

Eine Photovoltaik-Anlage besteht aus mehreren Komponenten: die Solarmodule erzeugen aus Sonnenlicht Gleichstrom, welcher von Wechselrichtern in netzkonforme Wechselspannung mit 230 V umgewandelt wird. über die Anzahl der einzelnen Solarmodule können Gesamtleistung und Flächenaufteilung frei variiert werden. Je nach Größe der Gesamtanlage können mehrere Wechselrichter zum Einsatz kommen. Die Montage der Solarmodule erfolgt auf einem Montagegestell, welches ebenfalls auf die speziellen Gegebenheiten ausgelegt wird.

Für die Zusammenstellung von Photovoltaik-Anlagen haben Sie die Möglichkeit, aus den Produkten verschiedener Hersteller zu wählen. Jeweils passend muss die Auswahl der technischen Details erfolgen, wie z.B. der Wechselrichter und des Montagegestells.

Eine komplette Photovoltaik-Anlage besteht aus folgenden Komponenten, für die wir für Sie weitere Informationen zusammengestellt haben:

  • Solarmodule
  • Wechselrichter
  • Montagegestell

Jede Einzelsituation benötigt eine spezielle Anpassung der Details, fragen Sie uns! Wir sind für Sie da, von der ersten Idee über die Planung bis hin zur Montage und Wartung.
